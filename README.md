# KYC / EDD Governance Monitoring Dashboard

An interactive **Power BI dashboard** developed to monitor **Know Your Customer (KYC)** compliance, **Enhanced Due Diligence (EDD)**, customer risk, and governance metrics. The dashboard provides actionable insights into customer risk profiles, overdue reviews, transaction anomalies, and compliance monitoring.

## 🎯 Project Overview

This dashboard simulates a Governance Monitoring solution used in financial institutions to support KYC and EDD processes. It enables users to monitor customer risk, identify compliance gaps, and visualize key governance metrics through interactive reports.

---

## 🚀 Features

- Interactive KPI cards for customer risk monitoring
- Customer risk classification (Low, Medium, High, Critical)
- Review overdue tracking
- Transaction anomaly analysis
- Expected vs. Actual transaction volume comparison
- EDD monitoring by risk category
- Interactive filtering and drill-down capabilities
